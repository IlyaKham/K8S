# Расчет ресурсов кластера Kubernetes ---  "Хамуро ИА"

## 1. Вводные данные

Проект состоит из следующих компонентов:

- **База данных** (StatefulSet) — отказоустойчивая, 3 копии.
- **Кеш (Redis)** (StatefulSet) — отказоустойчивый, 3 копии.
- **Фронтенд** (Deployment) — отдает статику, 5 копий.
- **Бекенд** (Deployment) — основная бизнес-логика, 10 копий.
